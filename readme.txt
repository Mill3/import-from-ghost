Import posts, images, tags and settings from a Ghost export file into Wordpress.

== Description ==
Generate an export file from your Ghost blog admin and use this plugin to import posts, their tags and a few translatable settings to your Wordpress installation. Images will also be copied across if you provide the URL for your Ghost blog.

== Installation ==
Upload the Ghost Importer plugin to your blog and activate it.
